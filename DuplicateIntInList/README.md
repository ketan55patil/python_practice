# Find duplicate integers from the list
* Accept integer input from the user
* Return the list of duplicate integers 

Sample output:
```
Enter an integer or q to quit:
1
2
q
User input: [1, 2]
List of duplicate integers: []
```
```
Enter an integer or q to quit:
1
2
1
q
User input: [1, 2, 1]
List of duplicate integers: [1]
```
